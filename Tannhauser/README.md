# Tannhäuser

Tannhäuser es una aplicación desarrollada en Python con Flet para la gestión de un supermercado inteligente. El sistema permite administrar usuarios, inventario, comentarios, actividad del sistema y sensores simulados, todo ello conectado actualmente a una base de datos MySQL.

## Funcionalidades principales

- Inicio de sesión con distintos roles
- Gestión de usuarios
- Gestión de inventario
- Panel de administrador
- Panel de superusuario
- Sistema de comentarios
- Registro de actividad del sistema
- Simulación de sensores
- Almacenamiento persistente en MySQL

## Tecnologías utilizadas

- Python
- Flet
- MySQL
- mysql-connector-python
- python-dotenv
